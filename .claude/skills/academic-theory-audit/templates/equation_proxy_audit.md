# 方程与守恒量审计模板

凡是涉及方程解释、诊断量物理意义、proxy 说法的地方，都必须填这一表。

## 审计条目
- 位置：
- 方程 / 诊断量：
- 文稿中的解释：
- 文稿援引的参考文献：

## 物理意义检查
1. 这个量的严格物理含义是什么？
2. 它属于哪一类？
   - 背景 -> 扰动
   - 扰动 -> 扰动（内部转换）
   - 源项 / 汇项
   - 其他
3. 这一定义依赖什么前提？
4. 文中是否清楚定义了符号、prime、overbar、背景场、平均方式？
5. 文中不同位置对同一量的描述是否前后一致？（如：一处说"气候态"，另一处说"纬向平均"——这是两个不同的定义）

## 能量库拓扑检查（关键）

当论文涉及能量预算时，必须画出流动图后逐一检查：

```
背景KE ──CK──→ 涡动KE
                    ↑
                  CP_vert（涡动APE → 涡动KE，内部转换）
                    |
背景APE ─CP_hor─→ 涡动APE ←──CQ── 非绝热加热
```

### 必须回答的问题：

1. **论文中被求和的各项，是否连接相同的能量库？**
   - 如果 A→B 和 B→C 被加在一起称为"从A到C的总转换"，这在物理上不成立。
   - 比喻：把"从银行取了多少钱"（背景→涡动APE）和"自己内部转了多少账"（涡动APE→涡动KE）加在一起说"银行给了我这么多"——不对。

2. **在涡动总能量预算（KE + APE）中，哪些项会被抵消？**
   - CP_vert 是涡动 APE → 涡动 KE 的内部转换。在涡动总能量预算中，它同时以正号出现在 KE 方程和负号出现在 APE 方程，自动抵消。
   - 如果论文把 CP_vert 当作涡动"从外部获取的能量"，就是错的。

3. **非绝热加热的正确诊断量是什么？**
   - 不是 ω'T'（那是绝热的 APE→KE 转换）
   - 而是 CQ ∝ Q'T'（Q = 非绝热加热率，T' = 温度扰动）
   - 如果论文把 ω'T' 称为"diabatic heating 的 proxy"，这是范畴混淆

4. **空间积分 vs 局地分布**
   - 局地分布有正有负，不代表净效果为正
   - 要证明"维持"，必须展示空间积分后的净值

## Proxy 检查
- 文稿是否把它当作另一个量的 proxy？
- 如果是，目标量是什么？
- 这种代理关系是：
  - 定义上等价
  - 文献严格支持
  - 经验近似
  - 不成立
- 如果不成立，正确应使用什么量？
- 引用是否真的支持这种 proxy 关系？（必须检查原文的精确论述）

## 典型失败模式
- 跨库求和：把不同能量库之间的转换项相加，赋予错误总解释
- 内部抵消忽略：把在总预算中会抵消的项当作净来源
- 范畴混淆：把绝热内部转换当作非绝热外部注入的 proxy
- 守恒/抵消关系错误：忽略了某项在完整预算中会被抵消
- 过度解释：文中赋予该量的物理含义超过了公式本身的支持
- 背景场定义不一致：一处说"气候态"，另一处说"纬向平均"
- 引用语义不匹配：引用说"A增强了B"，文稿写成"B是A的proxy"

## 审计结论
- 严重程度：Reject-level / Major / Minor / Adequate
- 必须如何改写：
- 原主张是否还能保留：可以 / 不可以 / 只能降级后保留
- 如果需要替代诊断量，具体是什么（变量名、计算方法、参考文献）
